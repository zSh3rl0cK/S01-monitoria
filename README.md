# S01 — Monitoria

Material utilizado durante as aulas de laboratório da disciplina **S01 (Paradigmas da Programação)** ao longo do período **2026/1**.

O repositório cobre os paradigmas **Orientado a Objetos** e **Funcional**, combinando explicações teóricas e exemplos práticos desenvolvidos aula a aula.

---

## O que você vai encontrar aqui

- **Slides em PDF** com o conteúdo teórico de cada aula — as gravações em vídeo são disponibilizadas separadamente apenas para os alunos da disciplina
- **Códigos de exemplo** comentados, desenvolvidos para ilustrar cada conceito no momento em que ele é introduzido
- **Exercícios práticos** com contextos variados, criados para fixar o que foi visto em aula

Os exemplos modelam situações do mundo real — e de ficção — com o objetivo de mostrar como os conceitos se aplicam na prática, indo além da teoria.

---

## Sobre os comentários

Todo o material prático está integralmente comentado, levando em conta que este pode ser o **primeiro contato** de alguns alunos tanto com as linguagens quanto com os paradigmas. Os comentários explicam não só *o que* o código faz, mas *por que* ele foi escrito daquela forma.

Se identificar qualquer erro ou inconsistência, entre em contato para correção imediata.
