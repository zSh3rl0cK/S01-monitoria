# S01-Monitoria

Material utilizado durante as aulas de laboratório da disciplina **S01 (Paradigmas da Programação)** ao longo do período **2025/2**.
O material aborda os paradigmas **Orientado a Objetos** e **Funcional**, combinando fundamentos teóricos e atividades práticas.

O conteúdo inclui:
* **Slides em PDF** com explicações teóricas (as gravações em vídeo são disponibilizadas apenas para os alunos da disciplina);
* **Códigos práticos** com temas diversos, desenvolvidos para aplicar os conceitos apresentados em cada aula.
  
Os exemplos de código modelam **entidades do mundo real** e exploram diferentes contextos, com o objetivo de demonstrar a **flexibilidade e aplicabilidade** dos paradigmas estudados e cada nova estrutura ou conceito abordado na aula.

## Observações

* Todo o material prático está **integralmente comentado** para facilitar a compreensão dos conceitos e das estruturas utilizadas nas linguagens, considerando que este pode ser o **primeiro contato** de alguns alunos com elas.
* Nem todos os comentários são autorais; porém, todos foram **revisados** para evitar erros.
* Caso identifique **erros ou inconsistências**, por gentileza, entre em contato para **correção imediata**.

### Estrutura do Repositório

O repositório está divido em relatórios que os alunos devem realizar semanalmente.
* Relatórios 5 a 11: Paradigma Orientado a Objetos.
* Relatórios 12 a 13: Paradigma Funcional.
